myFlix App

Movie Application

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints and Descriptions


Method EndPoint Response/Directions
GET /movies Returns JSON Object with a list of all top movies 
        {
          Title: String,
          Director: String,
          Genre: {
            Name: String,
            Description: String
          },
          Director: {
            Name: String,
            Bio: String
          }
          Featured: Boolean,
          ImageUrl: String
        }
GET /movies/[title] Input the movies title by name at the end of the URL where it says "[title]". If the movie's title contains spaces, replace it with a "%20", where the space would be instead.

JSON containing details about movie by title 
        {
          Title: String,
          Director: String,
          Genre: {
            Name: String,
            Description: String
          },
          Director: {
            Name: String,
            Bio: String
          }
          Featured: Boolean,
          ImageUrl: String
        }
GET /movies/genres/[title] Details about Genre of movie by the title of the movie. Do not input a specific genre by its name. If the movie's title contains spaces, replace it with a "%20", where the space would be instead.
GET /directors/[name] Details about director by name 
        {
          Name: String
          Bio: String
        }
POST /users Allows user to register 
        {
          Username: String,
          (required)
          Password: String,
          (required)
          Email: String,
          (required)
          Birthday: Date
        }
PUT /users/[Username] Allows user to update their information - will update information in JSON. 
                  {
                    Username: String, (required)
                    Password: String, (required)
                    Email: String, (required)
                    Birthday: Date (required)
                  }
POST /users/[Username]/movies/[MovieID] Allows user to add movie to their list of favorites
DELETE /users/[Username]/movies/[MovieID] Allows user to remove movie from their list of favorites
DELETE /users/[username] Allows user to delete account
POST /login In order to generate a token, you must navigate to the params. Under "Key", add a "Username" field and a "Password" field. Enter your personal information into the "Value" field next to the corresponding "Key" and send your request.

When done properly, this will return a JSON Object displaying your user information and token. 
                {
                    "user": {
                    "_id": String,
                    "Username": String,
                    "Password": String,
                    "Email": String,
                    "Birthday": Date,
                    "FavoriteMovies": [],
                    "__v": String
                  },
                    "token": String
                }
Now that you've logged in, you can copy and paste the key when necessary to send your requests. Navigate to any of the desired endpoints and find the Authorization header. Use the drop down and select "Bearer Token", upon doing so, you should see a "token" field that you can paste your token. Repeat this process for each endpoint in order to send your requests.
